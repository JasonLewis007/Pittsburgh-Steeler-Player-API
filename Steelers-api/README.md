# Steelers API

This is a simple API for managing Pittsburgh Steelers players using Node.js and Express.

## Endpoints

- GET /players
- GET /players/:id
- POST /players
- PUT /players/:id
- DELETE /players/:id

## Setup

```bash
git clone <your-repo-url>
cd <your-repo-name>/steelers-api
npm install
node src/index.js
